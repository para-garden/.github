# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any para-garden project, please report it by opening a private security advisory in the relevant repository.

1. Go to the repository's Security tab
2. Click "Report a vulnerability"
3. Provide details about the issue

We will respond within 48 hours and work with you to understand and address the issue.

## Scope

This policy applies to all repositories under the para-garden organization:

- [existence](https://github.com/para-garden/existence) - Text-based HTML5 game

## Supported Versions

We provide security updates for the latest release of each project. Older versions may not receive patches.
